# spigradle-sample

This is a multi-module sample project of [Spigradle](https://github.com/typst-io/spigradle/blob/master/docs/spigot_plugin.md) for Spigot plugin.

## Version Catalog

This project uses a [Version Catalog](https://docs.gradle.org/current/userguide/version_catalogs.html) configured in [libs.versions.toml](./gradle/libs.versions.toml).

- Spigot sample
  - [Groovy](spigot)
  - [Kotlin](spigot-kotlin)
- Bungeecord sample
  - [Groovy](bungeecord)
  - [Kotlin](bungeecord-kotlin)
- Nukkit sample
  - [Groovy](nukkit)
  - [Kotlin](nukkit-kotlin)
